graphes_deserts
===============

AMU - TP Complexite - Graphes Déserts


+--------------------------+
    Fonctionnement
+--------------------------+

** Les fichiers .c et .h sont dans le dossier /src
** Les fichiers exécutables sont dans le dossier /bin

** Pour compiler : make <nom_du_prog>
** Pour exécuter : ./bin/<nom_du_prog>
** Pour nettoyer : make clean
